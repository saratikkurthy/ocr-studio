# Phase 6E.2.5C — AI Research Copilot

This release adds a separate scholarly research workbench on top of the local manuscript index and Ollama integration.

## Features
- Research briefs
- Edition comparison
- Character profiles
- Place exploration
- Citation-backed timelines
- Theme analysis
- Workspace or collection scope
- Persistent saved reports
- Evidence dossier with page, project, document, language and OCR confidence
- Markdown, HTML and JSON export

## Local storage
- `.ocr-studio/research-reports.json`
- `.ocr-studio/research-exports/`

## Requirements
Build the manuscript index first and ensure Ollama is running with a selected local model.
