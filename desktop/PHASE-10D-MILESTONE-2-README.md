# Phase 10D Milestone 2 — Public Scholarly Web Portal & Reader

Adds a project-local publication builder that generates a deployable static scholarly website from OCR Studio project data.

## Features
- Portal site catalog and lifecycle
- Public page editor
- Responsive scholarly reader
- Client-side full-text search
- Structured JSON publication, page, and search-index endpoints
- Institutional branding and licence metadata
- Static/offline-friendly deployment folder
- Generation history and direct output opening

## Storage
`.ocr-studio/public-portal/` containing `sites.json`, `pages.json`, `analytics.json`, `settings.json`, and `exports/`.
