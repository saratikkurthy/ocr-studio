# Phase 6E.2.4 — Workspace Intelligence Dashboard

This release upgrades the home dashboard into a live mission-control view across an OCR Studio workspace.

## Included
- Workspace health score
- Project, document, collection, queue, OCR quality, review, publishing, duplicate and storage metrics
- Seven-day OCR throughput visualization
- Project health ranking and risk detection
- Recent cross-project activity
- Language portfolio
- Workspace selector and refresh
- JSON intelligence snapshots saved under `.ocr-studio/intelligence-snapshots`

## Data sources
The dashboard reads the existing local OCR Studio project files only. No cloud service or external analytics are required.
