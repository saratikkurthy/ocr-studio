# Phase 6E.2.6B — Review Workflow & Approvals

Adds a workspace-wide Review Workflow page that aggregates existing collaborative review assignments across recent projects.

## Features
- Cross-project review queue
- Status filters and free-text search
- Assignment status transitions: Assigned, In Progress, Needs Changes, Completed, Verified, Published
- Open-comment counters
- Priority badges
- Project navigation
- Per-project JSON review report export
- Workflow metrics and approval guide

Assignments and comments continue to use each project's existing OCR Studio review-collaboration store, preserving compatibility with the Project Review tab.
